# Thesis Game Build

> [!IMPORTANT]  
> The game name displayed in this build is not final and comes from a previous version. It may change in the final release.

This repository contains the Windows builds for my thesis project: **3D Role-Playing Game developed in Unity**. It provides the latest compiled version of the game for testing and demonstration purposes.

## Contents
- Windows builds
- Experimental/test builds

## About the Game
This is a 3D role-playing game developed as part of a computer science thesis. The project includes character movement, quests, and basic interactions within a 3D environment.

## How to Run
1. Download the latest release from [Releases](https://github.com/tothadam1188/thesis/releases/latest).
2. Extract the downloaded archive to a folder of your choice.
3. Navigate to the extracted folder.
4. Run the main executable (`Kingslayer.exe`).
